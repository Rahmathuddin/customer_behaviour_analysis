## Project Workflow

### 1. Data Loading

* Imported the dataset using Python.
* Performed initial inspection of rows, columns, and data types.
* Verified data quality and structure.

### 2. Exploratory Data Analysis (EDA)

* Generated summary statistics.
* Analyzed distributions and relationships.
* Identified trends, outliers, and missing values.
* Created visualizations for better understanding of the data.

### 3. Data Cleaning

* Handled missing values.
* Removed duplicate records.
* Corrected inconsistent formats.
* Treated outliers where necessary.
* Prepared a clean dataset for analysis.

### 4. SQL Analysis

* Loaded data into relational databases.
* Executed SQL queries using:

  * MySQL
  * SQL Server

  * Aggregations

  * Window functions
  * KPI calculations

### 5. Dashboard Development

* Built an interactive Power BI dashboard.
* Designed visualizations for key metrics and trends.

* Created a business-friendly dashboard layout.

### 6. Reporting

## How to Run

### Prerequisites

Install Python packages:

```bash
pip install pandas numpy matplotlib seaborn
```

### Run Analysis

```bash
python analysis.py
```

### Execute SQL Queries

1. Import the dataset into PostgreSQL, MySQL, or SQL Server.
2. Run the SQL scripts from the `sql/` folder.
3. Review query outputs and insights.

### Open Dashboard

1. Open the `.pbix` file in Power BI Desktop.
2. Refresh data connections if required.
3. Explore dashboard visuals and filters.
